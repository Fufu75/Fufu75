### About

Final-year MSc student in Big Data at ESME (Paris & Lyon), currently a research engineer intern at Astek.

**What I'm working on.** An LLM-assisted static analysis pipeline that detects, classifies and automatically fixes security vulnerabilities in source code. To benchmark it honestly I built a controlled synthetic dataset with injected vulnerability patterns and deterministic tests, so there's no data leakage — then ran 26 open-source LLMs through 4 prompting strategies across Python, Java and C.

**What I worked on before.** AI applied to architecture at KaAn Groupe in Geneva: image recognition, and an autonomous learning model for automated floor plan design.

**Outside the code.** I spent six months on exchange at the University of Canterbury in New Zealand, and a year running my school's student union — a €300k annual budget, a team to coordinate, and corporate partners to negotiate with.

**Learning right now.** Docker container orchestration and GitLab CI/CD pipelines.
